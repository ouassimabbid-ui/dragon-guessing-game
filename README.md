# Dragon Guessing Game (C)  : simple game that i made in my c learning journey 

A simple terminal-based number guessing game written in C.
## Description
The player must guess a randomly generated number between **1 and 100**.  
The game includes different difficulty levels that limit the number of attempts.w
it uses Standard Library (stdio.h, stdlib.h, time.h)
## Features
- Random number generation (1–100)
- Welcome menu with game rules
- Difficulty selection:
  - Easy (10 attempts)
  - Normal (7 attempts)
  - Hard (5 attempts)
- Hint system:
  - "Too high"
  - "Too low"
- Win / lose conditions
## How to run 
using any c compiler :)
